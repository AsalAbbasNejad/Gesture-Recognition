Gesture Recognition using TinyML on Arduino Nano 33 BLE SenseThis project demonstrates an end-to-end TinyML (Machine Learning on microcontrollers) workflow for classifying hand gestures (e.g., Punch and Flex) using an Arduino Nano 33 BLE Sense Rev2. The model is trained on IMU sensor data (accelerometer and gyroscope) and deployed for real-time, on-device inference.🚀 OverviewThe core objective is to move away from cloud-dependent processing by executing inference directly on the edge. This reduces latency, saves battery, and ensures data privacy.🛠 Hardware FeaturesThe Arduino Nano 33 BLE Sense Rev2 is equipped with:Microcontroller: Arm Cortex-M4 running at 64 MHz.IMU Sensor: BMI270 (6-axis accelerometer/gyroscope).Memory: 1 MB Flash and 256 KB RAM.📈 The WorkflowData Collection: Capturing raw IMU data via Arduino Serial Monitor and saving it to CSV files (punch.csv, flex.csv).Preprocessing: Normalizing raw sensor readings (Acceleration $\pm 4g$, Gyroscope $\pm 2000dps$) to a range between 0 and 1.Feature Extraction: Reducing high-dimensional input by calculating RMS and PSD (Power Spectral Density) to make the model lightweight for the microcontroller.Model Training: Building a Deep Neural Network (DNN) using TensorFlow/Keras in Google Colab.Conversion: Using the TensorFlow Lite Converter to generate a .tflite model, then converting it to a C source array (model.h).Deployment: Running the model on the device using the LiteRT (TensorFlow Lite Micro) library.🧠 Model ArchitectureInput Layer: 714 inputs (based on a 2-second sampling window).Hidden Layers: Two Dense layers (50 and 15 neurons) with ReLU activation.Output Layer: Softmax layer producing probabilities for "Punch" and "Flex" classes.📁 Repository Structuredata/: Contains the collected CSV sensor data.notebooks/: The Google Colab notebook used for training and conversion.arduino/: The final Arduino sketch, including model.h and the inference logic.🔧 Installation & UsageInstall the Arduino IDE or use the Arduino Cloud.Install the Arduino_BMI270_BMM150 and TensorFlowLite_Arduino libraries.Open the sketch in the arduino/ folder, ensure your board is connected, and hit Upload.Open the Serial Monitor to see real-time classification results!🎓 AcknowledgmentsThis project was developed as part of the Low-power Embedded Systems course, following the methodology provided by Prof. Kasım Sinan Yıldırım.
